Tomcat文件包含及读取漏洞（CVE-2020-1938漏洞复现）

2020/02/22
阅读时长: 1 分钟

0x01 漏洞影响

  • Apache Tomcat 9.x < 9.0.31
  • Apache Tomcat 8.x < 8.5.51
  • Apache Tomcat 7.x < 7.0.100
  • Apache Tomcat 6.x

0x02 前提

  • 目标机器开启AJP Connector 服务端口（默认开启，为8009端口）
